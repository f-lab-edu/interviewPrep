### 🧚‍♀️ 로컬 세팅 가이드
1) Repository에서 프로젝트를 다운 받습니다.
```
   git clone https://github.com/f-lab-edu/interviewPrep.git
```

2) node_modules를 설치합니다.
```
   npm install
```
4) 프로젝트를 실행합니다.
```
   npm start
```
